1. 사용자 입력 받기
- 사용자는 아이디(이메일 또는 닉네임)와 비밀번호를 입력한다.
- 프론트 단에서 비밀번호는 password 형식의 input을 통해 받는다.

2. 아이디와 비밀번호 확인
- 사용자가 입력한 아이디와 비밀번호가 데이터베이스에 저장된 값과 일치하는지 확인한다.

3. 로그인 성공 시
- 아이디와 비밀번호가 일치하면 로그인 성공 메시지를 반환하고, 사용자의 세션을 시작한다.
- (충돌을 위한 수정 사항) TOKEN을 반환한다.

4. 로그인 실패 시
- 아이디 또는 비밀번호가 일치하지 않으면 로그인 실패 메시지를 반환하고, 오류를 표시한다.
